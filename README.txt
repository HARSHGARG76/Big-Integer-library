All the arithmetic functions implemented accepts the parameters in the form of intal structure variable, making it easier for the client to go with the same interface for all the functions.
Input can be given with or without sign for any number. By default the number without sign is considered to be positive with (+) sign inserted in the starting of the string.
The return type of divide_intal() is long int which works for quotient upto range (-2,147,483,648 to 2,147,483,647).
